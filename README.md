# ⚡ Power Transmission Analysis Tool

Interactive Power System Simulator built using Streamlit.

## Features
- Power Circle Diagram (Sending & Receiving)
- Stability Analysis
- Stability Margin Calculation
- Load Slider (Real-time operation)
- Fault Simulation (Voltage Dip)
- PV Curve
- Voltage Profile
- Dark Professional Interface

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py
